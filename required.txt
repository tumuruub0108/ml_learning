Instance-Based Versus Model-Based Learning
One more way to categorize machine learning systems is by how they generalize. Most machine learning tasks are about making predictions. This means that given a number of training examples, the system needs to be able to make good predictions for (generalize to) examples it has never seen before.

There are two main approaches to generalization: instance-based learning and model-based learning.
